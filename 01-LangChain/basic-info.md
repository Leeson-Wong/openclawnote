# LangChain 项目 - 基本信息

## 项目来源
- GitHub: https://github.com/langchain-ai/langchain
- 官网: https://www.langchain.com/
- 许可证: MIT

## 项目简介
LangChain 是一个用于构建可靠 Agent 和 LLM 驱动应用程序的框架。它帮助开发者将可互操作的组件和第三方集成链接起来，简化 AI 应用程序开发，同时为底层技术的演进提供未来保障。

## 核心特点
1. **可靠 Agent 平台** - 专注于构建可靠的 AI Agent
2. **标准接口** - 为模型、嵌入、向量存储等提供统一接口
3. **模块化架构** - 基于组件的架构，快速原型开发
4. **丰富的集成** - 大量的第三方集成（模型提供商、工具、向量存储等）
5. **生产就绪** - 内置监控、评估和调试支持
6. **灵活的抽象层** - 从高级 chains 到低级组件

## 核心功能

### 实时数据增强
- 轻松连接 LLM 到各种数据源和外部/内部系统
- 庞大的集成库：模型提供商、工具、向量存储、检索器等

### 模型互操作性
- 轻松交换模型进行实验
- 适应行业前沿变化，抽象层保持开发连续性

### 快速原型开发
- 模块化、基于组件的架构
- 无需从头重建即可测试不同的方法和工作流

### 生产级功能
- 内置监控、评估和调试（通过 LangSmith 集成）
- 经过实战测试的模式和最佳实践

### 活跃的社区和生态系统
- 丰富的集成、模板和社区贡献组件
- 持续改进和最新 AI 发展

### 灵活的抽象层
- 从高级 chains（快速开始）到低级组件（细粒度控制）
- 随应用复杂性而成长

## LangChain 生态系统

### 核心产品
1. **LangChain** - 核心框架
2. **Deep Agents** (新) - 构建能规划、使用子代理和利用文件系统的复杂任务 Agent
3. **LangGraph** - 低级 Agent 编排框架，可靠处理复杂任务
4. **LangSmith** - Agent 评估和可观测性平台
5. **LangSmith Deployment** - 长运行、有状态工作流的部署平台
