# 消息管理 Skill

## 功能概述
这个技能涵盖了 OpenClaw 的核心消息管理功能，包括：

### 发送消息
- 多渠道发送支持
- 消息队列
- 批量发送
- 延迟发送
- 富文本支持
- 媒体附件
- 回复标记

### 接收消息
- 实时接收
- 消息解析
- 消息去重
- 消息验证
- 消息分类
- 多渠道聚合

### 消息重试
- 自动重试
- 重试策略配置
- 指数退避
- 重试上限
- 失败通知
- 手动重试

### 消息确认
- 自动确认
- 手动确认
- 确认类型
- 确认策略
- 批量确认
- 确认历史

### 消息策略控制
- 发送策略配置
- 优先级策略
- 渠道策略
- 重试策略
- 限流策略
- 审核策略

### 消息级别覆盖
- 级别设置
- 级别显示
- 级别筛选
- 级别路由
- 级别通知
- 级别覆盖

### 消息类型识别
- 自动识别
- 类型标记
- 类型处理
- 类型统计
- 类型过滤
- 自定义类型

### 消息路由
- 智能路由
- 渠道路由
- 用户路由
- 规则路由
- 路由优先级
- 路由日志

## 使用场景
- 消息发送
- 消息接收
- 消息重试
- 消息确认
- 消息策略管理

## 配置方法
通过 `/send` 控制消息发送策略
通过 `/retry` 管理消息重试
