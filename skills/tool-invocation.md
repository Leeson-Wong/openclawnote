# 工具调用 Skill

## 功能定位
这个技能定义了 OpenClaw 的工具调用系统，使用户能够通过自然语言触发各种外部工具和命令。

## 核心功能与操作指导

### 工具调用接口
- **调用方式**
  - 使用自然语言触发
  - 示例："运行 git status"

- **参数验证**
  - 自动验证输入参数
  - 实现：`ToolParameterValidator.check()`

### 内置工具
- **文件操作**
  - 支持读取、写入、编辑文件
  - 示例：
    ```json
    {
      "tool": "read",
      "args": {
        "file_path": "example.txt"
      }
    }
    ```

- **命令执行**
  - 支持 bash 命令
  - 示例：
    ```json
    {
      "tool": "exec",
      "args": {
        "command": "ls -la"
      }
    }
    ```

- **浏览器控制**
  - 使用 Playwright 控制浏览器
  - 示例：
    ```json
    {
      "tool": "browser",
      "args": {
        "action": "open",
        "url": "https://example.com"
      }
    }
    ```

### 工具分类
- **浏览器工具**
  - 包括：打开页面、截图、元素操作等

- **文件系统工具**
  - 包括：读取、写入、编辑文件

- **网络工具**
  - 包括：发送请求、下载文件等

- **系统工具**
  - 包括：执行命令、管理进程等

## 使用场景
- 自动化日常任务
- 执行系统命令
- 控制浏览器操作

## 注意事项
- 敏感操作需要确认
- 工具调用可能需要额外权限
- 监控工具执行日志
