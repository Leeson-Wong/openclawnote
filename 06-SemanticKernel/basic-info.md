# Semantic Kernel 项目 - 基本信息

## 项目来源
- GitHub: https://github.com/microsoft/semantic-kernel
- 组织: Microsoft

## 项目简介

Semantic Kernel 是 Microsoft 开发的轻量级 SDK，旨在将大型语言模型（LLMs）集成到传统的编程语言中。它提供了一种简单的方式，将 LLM 提示和 AI 功能编排为可复用的代码。

## 核心特点

1. **轻量级 SDK** - 简单、快速、高效
2. **多语言支持** - C#, Python, Java, JavaScript
3. **LLM 编排** - AI 功能的流程式编排
4. **插件系统** - 可扩展的插件架构
5. **企业级** - Microsoft 企业级质量和支持

## 技术栈

- **支持语言**: C#, Python, Java, JavaScript
- **LLM 集成**: OpenAI, Azure OpenAI, HuggingFace 等
- **许可证**: MIT

## 关键功能

### 1. Kernel（核心）
- 插件管理
- LLM 连接
- 函数执行
- 状态管理

### 2. Skills（能力）
- Semantic Function (语义函数）
- Planner (规划器）
- Memory (记忆）

### 3. Connectors（连接器）
- OpenAI
- Azure OpenAI
- HuggingFace
- 其他 LLM 提供商

## 快速开始

```python
pip install semantic-kernel
```

```csharp
dotnet add package Microsoft.SemanticKernel
```
