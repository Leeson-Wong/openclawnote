# LlamaIndex Agents 项目 - 基本信息

## 项目来源
- GitHub: https://github.com/run-llama/llama_index
- 组织: LlamaIndex

## 项目简介

LlamaIndex 是一个数据框架，用于构建基于 LLM 的应用程序。其中的 Agents 模块提供了 Agent 能力，专注于数据增强的 AI Agent，能够连接和查询各种数据源。

## 核心特点

1. **数据连接** - 连接各种数据源（文档、数据库、API）
2. **数据增强** - 为 LLM 提供相关的数据上下文
3. **Agent 能力** - 数据驱动的 Agent 系统
4. **检索增强生成（RAG）** - 强大的 RAG 实现
5. **多语言支持** - Python 和 TypeScript
6. **生态系统** - 丰富的数据连接器和索引

## 技术栈

- **语言**: Python, TypeScript/JavaScript
- **LLM 集成**: OpenAI, Anthropic, LLaMA, HuggingFace 等
- **向量数据库**: Chroma, Pinecone, Weaviate, Faiss 等
- **许可证**: Apache 2.0

## 核心概念

### Index（索引）
- 将数据转换为可查询的结构
- 向量索引、摘要索引、知识图谱索引
- 支持多种索引类型

### Query Engine（查询引擎）
- 高级查询能力
- 自然语言查询
- 多步推理和合成

### Agent（代理）
- 数据驱动的 Agent
- 查询和推理能力
- 工具使用能力
