# MemGPT 项目 - 基本信息

## 项目来源
- GitHub: https://github.com/cpacker/MemGPT
- 组织: MemGPT

## 项目简介

MemGPT 是一个具有持久记忆系统的 AI Agent 框架，专注于长期记忆和上下文管理，解决 LLM 的上下文窗口限制问题。

## 核心特点

1. **持久记忆** - 长期记忆系统
2. **层级记忆** - 短期、中期、长期记忆
3. **上下文管理** - 智能的上下文窗口管理
4. **记忆检索** - 相关记忆检索
5. **可扩展** - 可扩展的记忆后端

## 技术栈

- **语言**: Python
- **LLM 集成**: OpenAI, Anthropic
- **记忆存储**: 向量数据库、文件系统
- **许可证**: Apache 2.0

## 应用场景

1. **长期对话** - 需要长期记忆的对话系统
2. **个人助手** - 记住用户偏好的助手
3. **研究助手** - 研究过程中的记忆管理
4. **知识管理** - 个人知识库管理

## 核心优势

1. **记忆系统** - 强大的持久记忆
2. **层级设计** - 短期、中期、长期记忆
3. **智能检索** - 相关记忆检索
