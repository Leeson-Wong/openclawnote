# AgentGPT 项目 - 操作指南

## 快速开始

### 访问平台

AgentGPT 是一个基于 Web 的平台，通常提供：
1. 访问官方网站
2. 注册账户（如果需要）
3. 创建 Agent
4. 配置和运行

### 典型使用流程（基于推测）

```
1. 访问网站
   ↓
2. 创建 Agent
   ↓
3. 配置参数
   - Agent 名称
   - 目标描述
   - 模型选择
   - 最大迭代
   ↓
4. 启动执行
   ↓
5. 监控进度
   ↓
6. 查看结果
```
