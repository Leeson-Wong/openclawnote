# AgentGPT 项目 - 基本信息

## 项目来源
- GitHub: https://github.com/reworkd/AgentGPT
- 组织: Reworkd

## 项目简介

AgentGPT 是一个基于 Web 的 AI Agent 平台，允许用户创建、配置和部署自主 AI Agent。它将 AutoGPT 的概念带到了 Web，提供了现代化的用户界面和简化了 Agent 的创建和部署。

## 核心特点

1. **Web 界面** - 完全基于 Web 的用户体验
2. **无代码/Low-code** - 降低 Agent 创建门槛
3. **自主 Agent** - 自动执行任务
4. **实时部署** - 快速部署和测试
5. **多 Agent 管理** - 管理多个 Agent
6. **实时监控** - 监控 Agent 执行状态

## 技术栈

- **前端**: Next.js, React
- **后端**: Python（推测）
- **AI 集成**: OpenAI, Anthropic 等
- **部署**: Docker, Vercel（推测）
- **许可证**: 未明确

## 关键功能

### 1. Agent 创建
- 可视化配置界面
- 目标设置
- 模型选择
- 工具配置

### 2. Agent 执行
- 自主任务执行
- 迭代规划和执行
- 实时状态更新
- 结果展示

### 3. 监控和管理
- Agent 状态监控
- 执行日志查看
